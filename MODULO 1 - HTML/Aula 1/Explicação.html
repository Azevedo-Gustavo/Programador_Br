O html é o site por si só, a "base". O css é reponsável pela estilização do site. O javascript é responsável pelas interações do site. Tudo que altera o conteudo do site precisa do javascript. O site só irá abrir caso o servidor possuir o arquivo index.html. Caso este arquivo seja nomeado como qualquercoisa.html, ao informar o nome do site no buscador, será necessário informar o nome do arquivo para que o site abra. DNS: Domain Name Serve - servido de domínio. O DNS linka o nome do domínio (google.com) ao IP que está localizado no Site Server Center.
