ALINHAMENTO HORIZONTAL o alinhamento horizontal pode ocorrer ao utilizar a propriedade "margin: auto;" (normal ou com indicação do lado), nesta propriedade o elemento mantem o alinhamento informado independente do tamanho da tela. caso queira alinhar uma imagem através do "margin" é possível utilizando o display "block". mas caso haja necessidade de utilizar um display "inline-block" ou "inline". deve-se colocar a imagem dentro de uma div (criar um id/ class para ela) com a propriedade "text-align" e para o elemento "img" informar o display que se quer utilizar.
