OVERFLOW conteúdo é maior que o elemento. overflow: visible; - todo o conteúdo será visível, por mais que tenha ultrapassado as dimenssões do elemento. overflow: hidden; - todo conteúdo que ultrapassou o elemento ficará invisível porem, isso não é interessante pois "perdemos" informação. overflow: scroll; - o conteúdo passa "para dentro" do elemento e é gerado um scroll (x,y), mas o scroll só é habilitado para o lado que há necessidade o outro fica visível na tela mas não é funcional, o que acaba "poluindo" o layout. overflow: auto; - habilita apenas o eixo necessário para a visualização do elemento. overflow-y e overflow-x podem ser utilizados caso queira definir apenas um eixo para o scroll.
