POSITION - STATIC E RELATIVE STATIC mantém o padrão de display do elemento. RELATIVE será relativo ao display padrão a própria posição natural do elemento. caso o elemento "pai" altere de tamanho, o elemento que está sob efeito do "position: relative" não irá acompanha-lo e os elementos que vem depois dele ficão "suspensos", ficando um espaço vazio na tela. quando especificado, o elemento passa a poder se movimentar conforme as propriedades indicadas. a posição relativa só altera a posição do elemento indicado (top; bottom; right e left). portanto, a propriedade "margin" neste caso, não será útil.
