FORMULARIO o formulario é a integração entre o front-end e o back-end. o "method" dentro da tag "
" possui 2 opções o "Get" que irá buscar por algo dentro do servidor e que será informado na url e o "POST" que irá enviar alguma informação para o servidor e que não pode ser informada na url. a "action" será o domínio. TIPOS DE INPUTS text: input de texto checkbox: input de caixa de seleção *para diferenciar uma caixa de seleção da outra, é necessário utilizar o atributo "value" radio: input de botão *este input é utilizado quando se quer apenas 1 valor. para isso, é necessário que todos os inputs deste tipo tenho o atributo "name" igual. **caso queira que 1 dos botões seja marcado automaticamente, deve-se utilizar o atributo "checked". submit: botão de envio reset: botão de resetar SELECT a tag "select" cria uma barra de seleção (ex: estados). pode ser utilizado os atributos "size" para indicar quantos valores serão mostrados e este atributo criará uma barra de rolagem automaticamente. outro atributo é o "multiple" para que seja possível a seleção de mais de 1 valor. TEXTAREA usa-se "textarea" quando há a necessidade de ter um campo para comentários. ---------------------------------------------------- HTML 5 NOVOS INPUTS *necessário saber se o navegador utilizado suporta a nova feature do html 5. o site "caniuse" consegue ajudar com esta questão. INPUT - COLOR envia o hexadecimal de uma cor para o servidor. INPUT - DATE input de data, por padrão - yyyy-mm-dd. pode ser utilizado o input "datetime". *é possível definir um "max" e "min" para os valores. INPUT - FILE input de arquivo. INPUT - NUMBER input numérico. *é possível definir um "max" e "min" para os valores. INPUT - RANGE input de intervalo. *necessário definir um "max" e "min" para os valores. INPUT - TEL input "telefônico". INPUT - EMAIL só é enviado o e-mail caso o texto informado neste campo tenha a estrutura de um e-mail.
